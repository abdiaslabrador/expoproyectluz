Intruncciones de installación:
	1- Hacer clone
	2- Tener una versión de python en mi caso usé python 3.8
	3- Crear un entorno virtual y activarlo para instalar los paquetes necesarios.
	4- Instalar lo paquetes del requirement.txt ruteando donde esta la carpeta descargada desde la terminal ( cd .../pilatescenter), luego escribes "pip install -r requirements.txt". Esto instalará todos lo paquetes necesarios para correr el projecto.
	5- Ir a la carpeta "pilatescenter" y escribir python manage.py runserver
	6- Abrir el navegador y escribir en la barra de tu url http://127.0.0.1:8000/login o cualquier url que está en settings

	links disponibles hasta el momento después de activar el entor:
		http://127.0.0.1:8000/admin/
		http://127.0.0.1:8000/login/
		http://127.0.0.1:8000/logout/
		http://127.0.0.1:8000/plan/...
	  	http://127.0.0.1:8000/users/....
	
	7- Crear un elemento con name = 'Ninguno' en planes
	8- Crear elementos en los ejercicios
